# Assets bundling

To leverage code splitting (opens new window), lazy loading (opens new window) and tree shaking (opens new window), we favor module bundlers to manage our assets.

  • We prefer webpack (opens new window) to load and bundle our assets and dependencies together.
  • We use webpack-dev-server (opens new window) to serve our assets during development.
